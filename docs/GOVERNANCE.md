# Governance (Trumpality)

Trumpality is evidence-first and append-only.

## Non-negotiables
1. Statements are verbatim (no paraphrase).
2. Every statement/action must include a primary source URL.
3. No adjectives or editorial language inside record files.
4. Corrections are additive: create a new record that supersedes the old one.
5. If a claim uses fraud as justification, it must be handled comparatively in Executive_Rhetoric_Ledger.

## Allowed Sources (preferred order)
- Official transcripts / official video archives / Federal Register / court filings
- Major reputable journalism (AP, Reuters, etc.)
- Secondary sources only if they link primary sources

## Record Status
Use `status:` to indicate:
- `draft` (captured but not verified)
- `verified` (primary source checked)
- `superseded` (replaced by newer record)
